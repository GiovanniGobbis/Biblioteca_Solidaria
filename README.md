
# Biblioteca Solidária

Projeto criado como parte da atividade prática de HTML5.

## Estrutura
- `index.html` — página inicial com informações da organização.
- `projetos.html` — apresenta os projetos e formas de doação.
- `cadastro.html` — formulário com validação e máscaras.

## Tecnologias
- HTML5 semântico  
- CSS3 responsivo  
- JavaScript (máscaras e menu mobile)

## Como testar
1. Abra o projeto no Visual Studio Code.
2. Instale a extensão Live Server.
3. Clique com o botão direito em `index.html` → “Open with Live Server”.

## Envio no GitHub
1. Crie um repositório público chamado `biblioteca-solidaria`.
2. Envie todos os arquivos e pastas.
3. Copie o link do repositório.
